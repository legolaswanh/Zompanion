# 出生点配置说明

玩家出生点支持两类：**首次进入默认出生点**、**场景切换后传送入口出生点**。

---

## 1. 首次进入（HomeScene 起始出生点）

- **触发**：新游戏首次进入（如 OpeningStory → HomeScene）、或读档后首次进入某场景。
- **配置**：在 HomeScene 中放置一个 Tag 为 `SpawnPoint` 的 GameObject，作为默认出生位置。
- **行为**：玩家在该点位的 position 与 rotation 处出生。

---

## 2. 场景切换后出生（传送入口出生点）

- **触发**：玩家通过 `SceneExitTrigger` 进入目标场景时。
- **配置**：
  1. 在**来源场景**的 `SceneExitTrigger` 上配置 `Entry Point ID`（可选，留空则自动为 `From` + 当前场景名）。
  2. 在**目标场景**中放置 `SceneEntryPoint` 组件，`Entry Point ID` 与 SceneExitTrigger 中的一致。
- **行为**：
  - 出生位置 = 入口点 position + 偏移（左右朝向用 Offset X，上下朝向用 Offset Y）。
  - 玩家朝向 = 进入传送前的朝向（与 Animator 的 MoveX/MoveY 一致）。

---

## 3. 组件说明

### SceneExitTrigger（已有，新增字段）

| 字段 | 说明 |
|------|------|
| Target Scene Name | 目标场景名 |
| **Entry Point ID** | 目标场景中 `SceneEntryPoint` 的 ID，留空则使用 `From` + 当前场景名 |
| **Spawn Direction Override** | 进入新场景时的出生方向（用于偏移与朝向）。设为 (0,0) 则使用玩家 LastDir；建议按出口朝向填写，如右出口填 (1,0)，上出口填 (0,1) |
| Player Tag | 检测的玩家 Tag |

### SceneEntryPoint（新建）

| 字段 | 说明 |
|------|------|
| Entry Point Id | 入口 ID，需与 SceneExitTrigger 的 Entry Point ID 一致 |
| Offset X | 左右出口时沿朝向的偏移距离 |
| Offset Y | 上下出口时沿朝向的偏移距离 |

---

## 4. 配置示例

### HomeScene

- 保留默认 `SpawnPoint`（Tag: SpawnPoint）—— 首次进入时使用。
- 添加空物体 + `SceneEntryPoint`：
  - Entry Point Id: `FromOutDoorScene`
  - 放在庭院出口附近（玩家从 OutDoorScene 回到家的位置）。

### OutDoorScene

- 添加空物体 + `SceneEntryPoint`：
  - Entry Point Id: `FromHomeScene`（HomeScene 的出口 Trigger 指向本场景）
- 再添加：
  - Entry Point Id: `FromExplorationScene`（ExplorationScene 的出口 Trigger 指向本场景）

### ExplorationScene

- 添加空物体 + `SceneEntryPoint`：
  - Entry Point Id: `FromOutDoorScene`

### 各 SceneExitTrigger 的 Entry Point ID

| 场景 | 目标场景 | Entry Point ID（在 Inspector 填） |
|------|----------|----------------------------------|
| HomeScene | OutDoorScene | `FromHomeScene`（OutDoorScene 中对应 EntryPoint 的 ID） |
| OutDoorScene | HomeScene | `FromOutDoorScene` |
| OutDoorScene | ExplorationScene | `FromOutDoorScene` |
| ExplorationScene | OutDoorScene | `FromExplorationScene` |

---

## 5. 简化用法

若 `SceneExitTrigger` 不填 Entry Point ID，会自动使用 `From` + 当前场景名。  
例如：在 HomeScene 的出口 Trigger 上留空，则 OutDoorScene 中需有 Entry Point ID = `FromHomeScene` 的 `SceneEntryPoint`。

## 6. 出生方向偏斜问题

若发现玩家在新场景的出生位置总是偏移到错误方向，请使用 **Spawn Direction Override**：  
在 `SceneExitTrigger` 上按该出口的「进入新场景时的方向」填写，例如：
- 右出口（玩家从左边走进，进入新场景后应面朝右）：填 `(1, 0)`
- 左出口：填 `(-1, 0)`
- 上出口：填 `(0, 1)`
- 下出口：填 `(0, -1)`

留空 `(0, 0)` 时会使用玩家 LastDir，可能因停止移动、触发时机等导致方向不准。
